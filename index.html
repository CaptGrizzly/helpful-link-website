Recommended Settings

Steam Settings

  • FPS Counter: Click the Steam button (top left) > Settings > In-Game. Enable "In-game FPS counter" and toggle "High contrast color".
  • Xbox Controller: Connect via Bluetooth in Windows settings. In Steam, go to Settings > Controller and install the Xbox Extended Feature Support Driver.
  • Game Recording: Click Steam button (top left) > Settings > Game Recording. Enable "Record in Background" and set duration. Adjust shortcut keys.
  • Console Mode: Click the small TV icon (top right) labeled "Steam Big Picture" to enable a console-like experience. You may enter and exit this any time.
  • Adding Non-Steam Games: In Steam, go to "Games" (top left) > "Add a non-Steam game to my library", then select Minecraft Launcher (for example).
